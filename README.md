If anyone want to try in android, download https://obtainium.page/ and then from that program add app and paste https://github.com/MerApple/ChessOverMesh in url, that will make obtainium install chessmesh. 
You can download apk and exe directly from https://github.com/MerApple/ChessOverMesh/releases

This project also needs Meshtastic firmware running on your radio. You can download and flash it at https://flasher.meshtastic.org/
